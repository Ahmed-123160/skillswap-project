####Skill Swap Platform هي منصة لتبادل المهارات بين الناس. بدلاً من دفع فلوس لتعلم مهارة جديدة، يمكنك تبادل المهارات مع أشخاص آخرين.

مثال:يعرف التصميم يريد تعلم البرمجة → يتواصل مع شخص يعرف البرمجة يريد تعلم التصميم → يتبادلوا التعليم مجاناً

####Skill Swap Platform is a web application that enables users to exchange skills with each other. Instead of paying money to learn a new skill, you can barter your knowledge with others.

Example: A designer who wants to learn programming → connects with a developer who wants to learn design → they teach each other for free.
